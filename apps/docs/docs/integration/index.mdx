### Setup

1. Install and setup Homarr [according to the docs](https://homarr.vercel.app/docs/quick-start/)
2. Enable the Dashdot integration in the Homarr settings
3. Add your Dashdot instance to your services
4. You can then customize your widget directly in Homarr
